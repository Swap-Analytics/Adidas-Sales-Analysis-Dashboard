# 📊 Adidas Sales Analysis Dashboard

> An interactive Business Intelligence dashboard built using **Microsoft Power BI** to analyze Adidas sales performance across different regions, products, retailers, and time periods.

---

# 📌 Project Overview

The Adidas Sales Analysis Dashboard provides comprehensive insights into sales performance, operating profit, units sold, and regional trends. This dashboard enables business users to monitor KPIs, identify high-performing products, analyze retailer performance, and make data-driven business decisions through interactive visualizations.

---

# 🎯 Project Objectives

- Analyze overall sales performance.
- Monitor operating profit and profit margin.
- Track monthly sales trends.
- Compare sales across different regions and states.
- Identify top-performing products.
- Evaluate retailer-wise sales performance.
- Enable interactive business reporting using Power BI.
